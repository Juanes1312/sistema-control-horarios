Alcance

La tabla refleja el alcance del sistema y se divide en tres partes: entradas, procesos y salidas. Las entradas son toda la información necesaria para crear el sistema. Los procesos son las operaciones lógicas que gestionan distintos aspectos de la empresa. Las salidas son las funciones que desarrolla para resolver las gestiones necesarias.
Alcance del Sistema (Descripción Narrativa)
Lo que hace el sistema (Límites): El Sistema de Control de Horarios (SCH) automatiza el seguimiento de la jornada laboral en Virtual Llantas. Sus funciones principales son: gestionar perfiles y jornadas, registrar marcajes en tiempo real (entradas, salidas y recesos), administrar el flujo de solicitud y aprobación de permisos, y generar reportes consolidados de horas laboradas (pre-nómina) para facilitar la contabilidad.
Lo que NO hace el sistema (Fuera de alcance): Para delimitar el proyecto, el software no incluye:
•	Transferencias bancarias o pagos directos de nómina.
•	Procesos extra de Recursos Humanos (contratación, evaluaciones, etc.).
•	Cálculo y liquidación de impuestos o retenciones de ley.
•	Mantenimiento físico de equipos o redes.
Usuarios cubiertos: Es de uso interno exclusivo para cuatro roles: Administradores (configuración técnica), Director (gerencia), Coordinadores (gestión de equipos) y Empleados (marcajes diarios). Queda excluido cualquier actor externo como clientes o proveedores.
